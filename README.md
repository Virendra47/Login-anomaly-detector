# 🔐 Login Anomaly Detector

A security-focused application that detects unusual login activities by analyzing patterns such as location, device, login time, and user behavior. It helps identify suspicious access attempts and improves account security through anomaly detection.

## 🚀 Features
- Detect unusual login behavior
- Analyze login patterns and activity
- Identify suspicious access attempts
- Real-time anomaly alerts
- Simple and user-friendly interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Database

## 📦 Installation

1. Clone the repository
```bash
git clone <your-repo-url>
```

2. Open project directory
```bash
cd login-anomaly-detector
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Run the application
```bash
index.html
```

## 📊 How It Works
- Collect login activity data
- Analyze patterns such as time, device, and location
- Detect anomalies using predefined rules or ML models
- Generate alerts for suspicious logins
